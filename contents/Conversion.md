# Conversion

## Standard library
### Functions
| Functionality | C++ | Python |
| ------------- | --- | ------ |
| 字符序列转算术类型 | [`std::from_chars` (C++17)](https://en.cppreference.com/w/cpp/utility/from_chars.html) | `int`, `float` |
| 算术类型转字符串 | [`std::to_string` (C++11)](https://en.cppreference.com/w/cpp/string/basic_string/to_string.html) | `str` |
| 字符串转有符号整数 | [`std::stoi` (C++11), `std::stol` (C++11), `std::stoll` (C++11)](https://en.cppreference.com/w/cpp/string/basic_string/stol.html) | `int` |
| 字符串转无符号整数 | [`std::stoul` (C++11), `std::stoull` (C++11)](https://en.cppreference.com/w/cpp/string/basic_string/stoul.html) |
| 字符串转浮点数 | [`std::stof` (C++11), `std::stod` (C++11), `std::stold` (C++11)](https://en.cppreference.com/w/cpp/string/basic_string/stof.html) | `float` |
